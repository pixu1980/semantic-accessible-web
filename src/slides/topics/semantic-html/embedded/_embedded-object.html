<object></object>: Embeds external resources (e.g., images, videos, or interactive applications) with fallback content if the resource fails to load. Semantics: More flexible than <embed>, allowing fallback content.

object.html 
        
          <object
            data="family-guy.gif"
            type="image/gif"
            width="300"
            height="200"
          >
            <p>Your browser does not support this content.</p>
          </object>
