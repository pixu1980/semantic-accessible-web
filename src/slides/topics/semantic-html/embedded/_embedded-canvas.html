<canvas></canvas>: Provides a drawable area for rendering graphics via JavaScript (e.g., games, charts, or animations). Acts as a container; requires scripting to render content.

canvas.html 
        
          <canvas id="myCanvas" width="400" height="300"></canvas>
          <script>
              const canvas = document.getElementById('myCanvas');
              const ctx = canvas.getContext('2d');
              ctx.fillStyle = 'blue';
              ctx.fillRect(50, 50, 100, 100);
          </script>
